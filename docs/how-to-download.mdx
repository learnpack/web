### Downloading Exercises

1. Initialize LearnPack by typing the following command on your computer terminal:

```sh
$ learnpack install:exercises
```


2. You will be asked to select the language you wish the tutorial to be based on:

![alt](https://i.imgur.com/T2ZeRK3.png)

3. Next, you will select from the list of exercises avalaible to complete:

![alt](https://i.imgur.com/kwE7m1P.png)

LearnPack will now download the exercises selected.  They can be found in your workspace on a folder with the same name as the exercises you have just selected.


4. Following you will type the following command on your computer terminal to run the exercises that have just been downdloaded:

```sh
$ learnpack run
```

5. You may now click on the link provided in the terminal in order to launch the exercises.

![alt](https://i.imgur.com/k63n4eN.png)
