## Learn Pack Official Documentation

LearnPack is a learning package manager.  

It allows the user to create, publish, download and take different learning materials.

LearnPack can be used as a **student** or as a **teacher**.

To start using LearnPack as a student, please visit the *Quick Start for student* section.

To start using LearnPack as a teacher, please visit the *Quick Start for teacher* section.
